## ✨ Features

- **802.1X Policy Builder** — wired + wireless + remote access policy sets, conditions, results
- **TrustSec SGT / SGACL** — group tag assignment, segmentation policy matrix
- **Device Profiling** — endpoint classification with custom profiler conditions
- **Posture Assessment** — compliance checks, remediation, hardware/software/registry policies
- **Guest Portal Simulation** — self-registration, sponsor approval, hotspot guest flows
- **No Backend Required** — runs entirely in browser, free, no signup

## 🚀 Try It

```bash
git clone https://github.com/NETWORKERS-HOME-123/ise-simulator.git

npm install
npm run dev
```

Open `http://localhost:5173` and start practicing.
